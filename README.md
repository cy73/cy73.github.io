### 摘要🎆
GitHub个人主页，官方称呼是profile，是一个以markdown脚本语言编写的个人GitHub展示主页面，README文件后缀名为md；md是markdown的缩写，markdown是一种编辑博客的语言。.md 文件是 markdown 文件，使用的是 Markdown 语言。
如果你还不了解这个语言，可以去看 [Markdown 语法](#Markdown-语法)。
